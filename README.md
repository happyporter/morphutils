<pre>
- 검색어 1 : 올해 크리스마스에는

- 검색어 3 : 올해 크리스마스에는 눈이 내리지 않고 비교적 포근할 전망이다.


- 유사어 1 : 올해 크리스마스에는 눈이 내리지 않고 비교적 포근할 전망이다.(1.0)
